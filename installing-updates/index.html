Installing Updates

Installing updates to the platform's services is very easy. Log into the CAD server and run:

cd /silimate/platform-k8s/
sudo make recreate-ecr-secret
sudo make redeploy

This will fetch the latest Docker containers for the platform and deploy them on Kubernetes.
